# Ruboty::Kendama

Search kendama trick from youtube.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-kendama'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-kendama

## Usage

```
@ruboty kendama - Search a kendama trick randomly from Youtube.
```
